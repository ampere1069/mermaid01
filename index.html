流程說明

  1. 報廢需求
    使用單位/申請人提出資產或專案報廢需求。
  2. 報廢物件分類
    根據物件類型進行分類。
    A. 在建工程廢料再利用評估
    評估報廢物件是否可再利用於其他專案或用途。
    B. 固定資產類 (Fixed Assets)
    - 定義：每項成本超過新臺幣 20,000 元的資產
    - 辦公室設備及傢俱耐用年限：5年
    一般設備類
    - 建築物改良/裝潢
    - 工程測量設備
    - 業務展示設備
    - 辦公室設備
    - 個人電腦
    - 非個人電腦
    - 汽車
    資訊設備類
    B-1. 資訊類報廢 (IT Assets)
    - 定義：伺服器、電腦、網路設備等
    - 依據：《GITP-7 IT End User Policy》
    - 重點：銷毀機密資料，確保數據安全
    B-2. 遭竊遺失
    - 通知公司所屬部門與財務部門
    - 員警機關備案取得案件證明
    - 辦理報廢與銷帳
    - 流程結束
  3. 評估結果處理
    - 可再利用：進行資產轉移申請（使用 FAD-31 / FAD-33 表單）
    - 不可再利用：進行報廢申請（使用 FAD-35 或 FAD-18 表單）
  4. 採購部詢價
    針對報廢申請，採購部向外部廠商詢價。
  5. 填寫報廢費用審核表
    使用單位填寫報廢費用審核表（QAD-028）。
  6. 審核
    由專案經理、部門主管或總經理進行審核簽核。
  7. 執行報廢處理
    根據審核結果，執行變賣、資產轉移、銷毀或廢棄等處理。
  8. 歸檔結束
    完成所有程序後，將申請表單、前中後照片、審核文件CHECKLIST、處置證明等資料歸檔，流程結束。
